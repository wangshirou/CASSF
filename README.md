# CASSF
This repository provides the official PyTorch implementation of the paper:  **"A Class-Aware Semi-Supervised Framework for Semantic Segmentation of High-Resolution Remote Sensing Imagery"**, published in *IEEE Journal of Selected Topics in Applied Earth Observations and Remote Sensing*, 2025.  
[Paper Link](https://ieeexplore.ieee.org/document/11143061)
